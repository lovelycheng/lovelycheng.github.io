机器学习第一次编程作业

  1. 创建MATLAB的账户
  2. 点击斯坦福的证书
  3. 点击 ‘Access MATLAB Online’ 创建 在线matlab的账户

###上传编程作业

  1. 下载作业
  2. 登录matlab online
  3. 输入如下命令，并运行：
    • unzip machine-learning-exn.zip

###编程作业

  1. ex1.m - Octave/MATLAB script that steps you through the exercise
  2. ex1 multi.m - Octave/MATLAB script for the later parts of the exercise
  3. ex1data1.txt - Dataset for linear regression with one variable
  4. ex1data2.txt - Dataset for linear regression with multiple variables
  5. submit.m - Submission script that sends your solutions to our servers

###need to complete files:

  • [?] warmUpExercise.m - Simple example function in Octave/MATLAB
  • [?] plotData.m - Function to display the dataset
  • [?] computeCost.m - Function to compute the cost of linear regression
  • [?] gradientDescent.m - Function to run gradient descent
  • [y] computeCostMulti.m - Cost function for multiple variables
  • [y] gradientDescentMulti.m - Gradient descent for multiple variables
  • [y] featureNormalize.m - Function to normalize features
  • [y] normalEqn.m - Function to compute the normal equations
