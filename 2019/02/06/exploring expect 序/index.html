exploring expect 序

Expect 正在快速的演变成unix用户的工具箱中的一员. 它允许你自动化的使用Telnet,FTP,passwd,rlogin,以及上百种其他的需要使用人工交互的应用。

使用 Expect 自动化那些应用将使你减少任务所需时间，在一些情况下，也能让你能够解决以前从来没有考虑过的新问题。 比方说，你能够使用Expect 去测试交互程序而不用去改他们的接口。或者包装类似Motif的前端交互程序去使用 按钮，滚动条，或者其他图形界面的元素去控制应用，而不需要重新编译源程序。

当Expect 和 第三方程序调用时，甚至不需要源码。使用Expect能够将互联网应用集成在一起。包括Telnet Archie FTP Gopher 和 Mosaic.

Don Libes 是Expect的创造者也是本书的作者。在本书中，他提供了全面的视角，介绍了Expect的所有特性，允许你立即在你的项目实践Expect。

他以脚踏实地和幽默的方式提供了大量具有挑战性的实际应用程序示例，以及如何使用Expect自动化它们以节省您的时间和金钱。

Expect 是基于TCL衍生的第一种新的程序，工具命令语言（TCL）正在影响(Rock)计算机科学社区。

这本书也提供了TCL的简介，并且解释了Expect如何在交互自动化方面应用了Tcl的功能。

无论你的兴趣是Expect或者是交互自动化，或者是简单学习Tcl和tcl是如何应用在真实的软件中，你都会发现Exploring Expect 是一个易于理解并且有价值的信息宝库。
